## Integrazione WordPress: plugin e REST API

Per chi cerca "Come automatizzare la scrittura dei blog Automatik Blog", l’integrazione con WordPress è centrale: pubblica articoli, immagini e meta SEO con pochi clic. <a href="https://developer.wordpress.org/rest-api/">Manuale ufficiale REST API WordPress</a>

| Metodo | Vantaggi | Livello tecnico |
|---|---:|---|
| **Plugin** | Installazione semplice, interfaccia grafica | Basso |
| **REST API** | Flessibile, personalizzabile | Medio-Alto |

Setup rapido:
1. Installare il **plugin** ufficiale o abilitare la **REST API**. <a href="https://developer.wordpress.com/docs/api/namespaces-versions/">Distinzione namespace REST API WordPress.com</a>  
2. Creare una chiave API con ruolo limitato.  
3. Mappare campi (titolo, contenuto, immagine, meta).  
4. Testare con un articolo prova e attivare la pubblicazione programmata.

Sicurezza e backup:
- Ruoli API con permessi minimi.
- Autenticazione via token su **HTTPS**.
- Backup giornalieri e test mensile di ripristino.
- Log conservati (es. 30 giorni).

## Immagini AI e Web Stories: contenuti visuali automatizzati

Automatik Blog trasforma articoli in contenuti visuali ottimizzati: immagini AI coerenti con testo e **Web Stories** pronte per social e sito.

### Generare immagini AI coerenti
- Prompt brevi e mirati con palette e stile del brand.
- 3 varianti per immagine, scelta e ottimizzazione. <a href="https://www.w3.org/WAI/tutorials/images/">Linee guida alt-text per immagini accessibili</a>
- Salvataggio di preset per uniformità.

### Creazione automatica di Web Stories
- Template modulari: copertina, slide, CTA. <a href="https://wp.stories.google/docs/best-practices/">Best practices per Web Stories e ottimizzazione visiva</a>
- Testo breve, immagini verticali 9:16, metadati SEO.
- Pubblicazione automatica su WordPress e canali social.

Formati e performance:
- Preferire **WebP/AVIF** per il web, 1080x1920 px per Web Stories.
- Usare srcset, compressione e lazy loading per velocità.
